1. Difference between Frontend, Backend, and Full-Stack Development

Frontend: It is the part of a website users interact with directly. It includes everything a user sees â€” like buttons, colors, text, and layout. Example: When you open Amazon, the product listings, search bar, and navigation menu are built using HTML, CSS, and JavaScript.

Backend: It manages the logic, database, and server operations behind the scenes. For example, when you log in to Amazon, the backend verifies your credentials and fetches your order data. Technologies include Node.js, Python, PHP, etc.

Full-Stack: A developer who works on both frontend and backend. Example: A developer who designs the Amazon UI and also handles server-side logic using Node.js and databases like MongoDB.
